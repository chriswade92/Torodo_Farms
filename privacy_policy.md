# Politique de Confidentialité - TorodoFarms

Dernière mise à jour : Mars 2024

## Introduction

TorodoFarms s'engage à protéger la vie privée des utilisateurs de notre application mobile. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations.

## Informations Collectées

Notre application collecte les informations suivantes :
- Historique des ventes
- Données de transaction
- Informations sur les produits commandés

Nous ne collectons aucune information personnelle identifiable.

## Utilisation des Données

Les données collectées sont utilisées uniquement pour :
- Gérer les commandes
- Suivre l'historique des ventes
- Améliorer l'expérience utilisateur

## Stockage des Données

Toutes les données sont stockées localement sur votre appareil. Nous n'utilisons pas de serveurs externes pour stocker vos informations.

## Sécurité

Nous prenons des mesures de sécurité appropriées pour protéger vos données contre l'accès non autorisé ou la modification.
